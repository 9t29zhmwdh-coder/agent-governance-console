This repository was scaffolded from an internal template and subsequently reviewed and
curated by the maintainers. Automation was used to reduce boilerplate; all content has
been manually reviewed. For details about the template, generation process, and review

## Automatisierung und Review

Dieses Repository wurde aus einem internen Template gescaffoldet und anschliessend von
den Maintainerinnen und Maintainern geprüft und angepasst. Automatisierte Hilfsmittel
wurden eingesetzt, um Boilerplate zu reduzieren; alle Inhalte wurden manuell überprüft.

- `ARCHITECTURE.md`: system architecture and data flow
- `PRIVACY.md`: privacy policy (opt-in telemetry, no default data egress)
- `ROADMAP.md`: planned milestones v0.1 through v1.0
- `CODE_OF_CONDUCT.md`: contributor covenant
- `SECURITY.md`: vulnerability reporting policy
- `CONTRIBUTING.md`: contribution guidelines (incl. audit-log immutability rule)
- `CHANGELOG.md`: initial v0.1.0 entry
- `RELEASES.md`: release process guide
- `SKELETON.md`: canonical file tree and CI reference
- `agc-core/src/`: trace, policy, audit, telemetry modules with unit tests
- `agc-api/src/main.rs`: Axum REST server stub
- `examples/`: working usage examples
- `scripts/`: Azure setup and audit export scripts
- `docs/`: Azure integration, trace schema, policy DSL, API reference
- `.github/workflows/ci.yml`
- `.github/ISSUE_TEMPLATE/`
- `.github/PULL_REQUEST_TEMPLATE.md`

All files have been reviewed and adjusted to fit the project context.
